## [¬º-°]¬ Informations importantes :

### Versions utilisées :
- PHP : 8.2.4
- Serveur : 10.4.28-MariaDB - mariadb.org binary distribution
- Interclassement DB : utf8mb4_general_ci
- Ajax : https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js
- Polices googles fonts
- Icons Font Awesome

### Hierarchie du projet :
- Dossier img: contient les images principales, de posts, du menu ainsi que des personnages dans differents dossiers 
- Dossier js : continent un fichier avec des fonctions JavaScript
- Dossier pages :  Contient les differentes pages et bout de pages de l'interface
- Dossier php : contient toute les requettes (excepté les select qui sont dans les pages)
- Dossier style : Contient toute les feuilles de style permettant de styliser les pages

### Mise en place du projet:
- Recuperer le projet et le decompresser
- Creer une base de donnée au format utf8mb4_general_ci avec PHPMYADMIN
- Importer le fichier BENEDUCI.sql à cette derniere: 6 tables doivent se creer (users, posts, likes, forum, commentaires, catégories). Ces tables sont déjà remplies.
- Inserer le dossier BENEDUCI_SOLO dans le localhost
- Changer les identifiants de connexion pour la base de données dans le dossier assets, dossier php, fichier connectDb.php
- Aller sur le navigateur et lancer le projet sur le localhost

N.B. Ce projet à été réalisé en utilisant XAMPP mais peut etre lancé avec n'import quel type de plateforme de serveur web
